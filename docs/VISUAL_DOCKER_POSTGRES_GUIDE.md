## Code Example: How Automatic Saving Works

### What You Write (Python Only):
```python
# app/api/routes/query.py

@router.post("/query")
async def process_medical_query(
    request: MedicalQueryRequest,
    current_user: User = Depends(get_current_user_optional),
    db: Session = Depends(get_db),
):
    # Process the query
    result = await workflow.process_query(request.text)
    
    # â­ THIS LINE SAVES TO DATABASE AUTOMATICALLY â­
    await QueryService.create_query_record(
        db=db,
        user_id=current_user.id if current_user else None,
        query_text=request.text,
        response_text=result["response"],
        processing_time=1.5,
        success=True
    )
    # That's it! No SQL written! âœ…
    
    return response
```

### What SQLAlchemy Does Behind the Scenes:
```sql
-- You NEVER write this, SQLAlchemy does it automatically:
INSERT INTO queries (
    id, user_id, query_text, response_text, 
    processing_time, success, timestamp
) VALUES (
    'uuid-12345', 'user-uuid', 'What is diabetes?', 
    'Diabetes is...', 1.5, true, '2025-10-28 10:30:00'
);
```
